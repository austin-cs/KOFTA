    \textbf{Abstract} \\[0.5cm]

\normalsize 
  
Fuzzing is a critical technique in the field of information security. However, most existing fuzzing tools focus exclusively on mutating input files or standard inputs with fixed command-line options, limiting their flexibility in exploring multiple options (multi-arguments) and often requiring expert-defined option configuration files. To address these limitations, this study proposes an option-aware fuzzing framework called KOFTA. KOFTA integrates static and dynamic analyses to automatically explore available command-line options and parameters, employing taint inference techniques to precisely identify critical parameters that influence branching paths, thereby fully exploring extensive input spaces. Additionally, by enhancing AFL's forkserver, KOFTA dynamically updates the target program's command-line arguments, significantly reducing the overhead of system calls during testing. Experimental results demonstrate KOFTA's superior performance compared to traditional multi-argument fuzzing tools in terms of improved test coverage, discovery of undocumented options and parameters, and vulnerability detection. An open-source prototype is provided to support future research and practical applications.
